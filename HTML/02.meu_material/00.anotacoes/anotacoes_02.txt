Formatação:

Tags (HTML):

Descontinuadas:

<u> : subinhado.

Usuais:

<b> : Negrito.

<i> : Itálico.

<em> : Ênfase.

<del> : Texto Riscado.

<center> : Alinhamento Central.

<span> : Formatar pequenos trechos de código (Geralmente acompanhado pelo atributo "style").

Atributos (HTML):

Alinhamento : align="left/right/center/justify"

Formatar : style=""

CSS:

Sublinhado = text-decoration: underline;

Nível do Negrito = font-weight: normal/bold/bolder/[valor específico];

Alinhamento = text-align: left/right/center/justify(Ajusta o texto para ficar a esquerda, mas também a direita. Uniformimente espalhado.);
